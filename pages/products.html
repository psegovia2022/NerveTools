Small Manipulator

This device has been used to record extra cellular action potentials from single unit cells in the prefrontal cortex and hippocampus (Negrón-Oyarzo et al. 2018).

You can find a detailed description of this manipulator in both publications.

If you download and use it, please cite the oublication that better match your research.

We have successfully printed the parts of this device using "smooth fine detail plastic," "white processed versatile" and "black Hi-Def acrylate."
